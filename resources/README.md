# Course 4 Resources

Supplementary materials for Generative Deep Learning.

## Cheat Sheets

- Model families: VAE, GAN, autoregressive, flow, EBM, diffusion, transformer
- Diffusion sampling: schedule, guidance scale, steps, seed, negative prompt, decoder
- Transformer systems: prompting, RAG, LoRA/QLoRA, DPO/RLHF, KV cache, quantization
- Evaluation: fidelity, diversity, grounding, safety, latency, cost, human review

## Tooling Notes

- TensorFlow/Keras follows Foster's book code path.
- Hugging Face `diffusers`, `transformers`, `peft`, and evaluation harnesses are useful modern supplements.
- Keep generated samples with prompts, seeds, model IDs, scheduler/config, and postprocessing steps.
